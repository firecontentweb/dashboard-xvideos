## O que é isso?

Um painel web onde você faz upload do CSV exportado do XVideos e visualiza instantaneamente:

- 💰 Ganho total do período
- 📅 Média de ganho por dia
- 👥 Crescimento de memberships
- 📈 Gráfico de ganhos ao longo do tempo
- 📊 Comparativo mensal

**Análises avançadas** como frequência de posts, insights estratégicos, projeções e análise de virais estão disponíveis via consultoria com a [Fire Content Web](https://wa.me/447460364671).

---

## Como usar

**1. Exporte seu CSV do XVideos**
- Acesse seu painel no XVideos
- Vá em **Statistics**
- Clique em **Export CSV**
- ⚠️ Nunca abra o arquivo no Excel antes — isso corrompe os dados

**2. Acesse o painel**
- Acesse o link do app
- Arraste o arquivo `.csv` para a área indicada
- Veja suas análises em segundos
